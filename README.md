This is a tdd sample repository

## Requirement

* Node.js 14+

## Installation

```shell
npm install
```

## Running unit tests

```shell
npm test
```

## Running lint

### Check with lint

```shell
npm run lint
```

### Fix it with lint

```shell
npm run lint:fix
```
